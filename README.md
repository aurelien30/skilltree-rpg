# SkillTree RPG

Un jeu de gestion de compétences inspiré des arbres de talents des RPG.

## Technologies utilisées

- **Frontend** : React + Vite
- **Backend** : Node.js + Express
- **Base de données** : SQLite (simple et sans configuration)

## Installation

### Prérequis

- Node.js version 18 ou supérieure

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Structure du projet

```
skilltree-rpg/
├── frontend/          # Application React
├── backend/           # API Node.js
└── README.md
```
